# minesweeper

Functional MineSweeper game made in Scala.

Current functionalities:
- Reveal block with LMouseClick.
- Flag block by holding space + LMouseClick.
- Bombs are spread at random.
- Score tracker.
- Algorithm for revealing adjacent blocks.
