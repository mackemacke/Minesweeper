# Minesweeper

This repository contains the source code for a functional Minesweeper game made in Scala.

## Current functionalities
- __Reveal Block__: Use Left Mouse Click.
- __Flag Block__: Hold Space + Left Mouse Click.
- __Random Bomb Placement__: Bombs are spread randomly.
- __Score Tracker__: Keeps track of the player's score.
- __Reveal Algorithm__: Efficiently reveals adjacent blocks.
